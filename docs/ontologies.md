# Ontologies

Initial assessments are underway - check back soon for the first results.

## Looking for specific ontologies?
Explore the [NFDI4Energy Collection](https://terminology.tib.eu/ts/ontologies?collection=NFDI4Energy)
in the [TIB Terminology Service](https://terminology.tib.eu).<br>
The collection is open for contributions, feel free to suggest an ontology to add to this collection.
