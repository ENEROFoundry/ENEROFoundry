Introduction
============
| The OEO Foundry is a collaborative initiative for developing interoperable ontologies in the energy domain.
| It follows shared principles to ensure high-quality and FAIR (Findable, Accessible, Interoperable, and Reusable) ontologies.
| The mission is to develop a family of interoperable ontologies that are both logically well-formed and scientifically accurate.

Documentation
=============
| The website is created with Markdown using `MkDocs <https://www.mkdocs.org/>`_ and `mike <https://github.com/jimporter/mike>`_.
| All files are stored in the ``docs`` folder of the repository.
| A **GitHub Actions** deploys the ``develop`` branch on a **GitHub Page**.
| The documentation page is: `oeofoundry.github.io/website/ <https://oeofoundry.github.io/website/>`_
